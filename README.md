# 서울 문화시설 3D 지리공간 연구 플랫폼

서울특별시 전역의 실제 지형·건축·교통·문화시설 데이터를 기반으로 구축한
**인터랙티브 3D 문화도시 연구 프로토타입**입니다.
관광용 시각화가 아니라 문화시설의 입지·접근성·지역 불균형 연구를 위한 도구입니다.

## 실행 방법

```
cd seoul-culture-3d
python -m http.server 8123
```

브라우저에서 http://localhost:8123 접속. (인터넷 연결 필요 — 지도 타일·지형 타일을 실시간 로드)

## 구성 데이터 (전부 실측·공개 데이터, 임의 생성 없음)

| 레이어 | 내용 | 출처 | 기준일 |
|---|---|---|---|
| 지형 (DEM) | 실측 표고 — 북한산·관악산·남산 등 실제 능선/경사 | AWS Terrain Tiles (SRTM 계열, terrarium 인코딩) | 타일 서비스 최신 |
| 건물 | 서울 전역 실제 건물 윤곽 + 높이 3D 압출 (LOD1) | OpenStreetMap / OpenFreeMap 벡터타일 | 실시간 |
| 도로·철도·하천·한강·교량 | OSM 기반 실제 선형 | OpenStreetMap | 실시간 |
| 문화시설 | **2,395개소**, 15개 유형 분류 | OSM(2026-07) + 문체부 2025 문화기반시설총람(2025-01-01, 공공누리 1유형) | 병기 |
| 지하철역 | 298개 역 지점 | OpenStreetMap | 2026-07-02 |
| 자치구 경계 | 25개 자치구 | 통계청/서울시 (southkorea/seoul-maps) | 2013 (경계 실질 불변) |
| 인구·면적 | 자치구별 참고치 | 행정안전부 주민등록인구 **근사치(2024, 확인 필요)** / 공식 행정구역 면적 | 2024 |

좌표계: **WGS84 (EPSG:4326)**. 모든 레이어 동일 기준으로 정합.

## 시설 유형 분류 (OSM 태그 → 연구 분류)

공연장 670 · 공공도서관 609 · 박물관 280 · 복합문화공간 184 · 기념관·기념물 178 ·
갤러리·전시공간 155 · 영화관·시네마테크 105 · 미술관 67 · 문화원·지역문화센터 53 ·
역사유적·문화재 29 · 작은도서관 26 · 궁궐·전통문화시설 24 · 콘서트홀 10 · 국악공연시설 5

**문체부 2025 총람 병합**: 총람 서울 시설 496건 중 185건은 기존 OSM 시설의 속성 보강
(운영주체·국공사립·개관연도·연락처), 246건 신규 추가(Nominatim 지오코딩, 근사 좌표),
65건은 지오코딩 실패로 지도 제외(명단은 scripts 로그 참조). 시설별 출처·기준일은 팝업에 표기.

이름 없는 객체 제외, 동일 명칭 300m 이내 중복 제거, 서울 경계 밖 제외.

### 스페셜티 카페 (참고 레이어, 152개소)

OSM 카페 3,834곳 중 다음 기준으로 선별. "예술적 분위기"는 데이터로 검증 불가하므로
**검증 가능한 대리 기준**을 쓰고, 각 시설 팝업에 선별 근거를 명시:

1. 대형 프랜차이즈 제외 (스타벅스·커피빈·이디야·메가·컴포즈 등 상호/브랜드 태그, 1,039곳 배제)
2. 포함 기준(하나 이상): 스페셜티 브랜드 상호(프릳츠·앤트러사이트·블루보틀·나무사이로 등) /
   로스터리 태그(craft=coffee_roaster) / 로스팅·드립·브루잉·커피랩 계열 상호 키워드 /
   원두전문점 태그(shop=coffee)
3. **문화소외·격자 밀도 분석에서는 제외** — 문화시설 통계의 무결성 유지, 지도 참고 레이어로만 표시

## 기능

- **현실 모드 / 분석 모드 / 야간 모드**
- 3D 지형 강조 슬라이더, 건물 투명도, 시설 표식 크기 조절
- 시설 클릭 → 명칭·유형·주소·운영주체·공공민간·설립연도·입장료·휠체어 접근·
  최근접 지하철역과 거리·웹사이트·데이터 출처 팝업 (미확인 항목은 **"자료 없음"** 표기)
- 검색(시설명) · 필터(유형 14종, 자치구, 공공/민간)
- 분석 레이어: 커널 밀도 열지도 · 500m 격자 밀도 · **문화소외 후보 격자**(최근접 시설 직선 800m 초과, 384셀) · 자치구 단계구분도(시설수/인구 1만명당/면적 1㎢당)
- 대표 장면 7종 (서울 전체 조감 / 역사도심 / 한강 문화축 / 강남 / 서북권 창작축 / 동북권 / 문화소외 분석)
- 자치구 통계 패널(유형별·자치구별 막대그래프, 공급 지표 상·하위)
- 두 자치구 비교표
- 필터 적용된 데이터 **GeoJSON / CSV 내려받기**
- 범례·축척·방위(나침반)·데이터 기준일 상시 표시

## 산출물

```
data/facilities.geojson        시설 위치·속성 (EPSG:4326)
data/grid500.geojson           500m 격자 밀도 + 문화소외 플래그
data/subway.geojson            지하철역
data/districts.geojson         자치구 경계
data/stats_by_district.json    자치구별 통계
output/facilities.csv          시설 전체 CSV (UTF-8 BOM, 엑셀 호환)
output/stats_by_district.csv   자치구 통계 CSV
output/facilities.geojson      시설 GeoJSON 사본
output/grid500_density.geojson 격자 밀도 GeoJSON 사본
scripts/process.py             데이터 처리 파이프라인 (재실행으로 갱신 가능)
scripts/overpass_*.txt         원자료 수집 쿼리 (Overpass QL)
```

갱신 절차: `overpass_*.txt` 쿼리 재실행 → `raw_*.json` 교체 → `python scripts/process.py`.
데이터 계층·분석 계층·시각화 계층이 분리되어 있어 전체 재구축 없이 갱신됩니다.

## 초기 분석 결과 (자치구별 시설 수, OSM 2026-07 기준)

종로구가 539개소로 압도적 1위 — 역사도심(궁궐·갤러리·공연장 밀집)의 집중이 데이터로 확인됨.
문화소외 후보 격자 442셀은 주로 외곽 주거지역·산악 인접 지역에 분포.
상세 수치는 `output/stats_by_district.csv` 참조.

## 한계와 정직한 표기 (데이터 정확성 원칙)

1. **방문객 수·운영 예산·객석 수·프로그램 수**: OSM에 없는 항목 — 임의 생성하지 않고 "자료 없음" 표기.
   서울열린데이터광장·문화체육관광부 문화기반시설총람 API 키 연동 시 확장 가능.
2. **지하철역 거리는 직선거리**입니다. 네트워크 보행거리·대중교통 이동시간(등시선) 분석은
   OSRM/Valhalla 라우팅 엔진 연동이 필요하며 현재 버전에는 포함되지 않음.
3. **문화소외 격자**는 직선거리 800m 기준의 근사 지표 — 정책 판단에는 네트워크 분석 병행 필요.
4. 인구는 주민등록인구 근사치(2024)로, 통계청 확정치로 교체 권장.
5. OSM 커뮤니티 데이터 특성상 소규모 민간 공간(대안공간·작은도서관)은 과소 등록 가능.
6. 시간 슬라이더(1990~현재 개관·폐관 변화)는 연도별 원자료 부재로 미구현 —
   과거 자료를 임의 변환하지 않는다는 원칙에 따름.
7. 건물은 LOD1(윤곽+높이 압출) 수준. 주요 랜드마크 LOD3 정밀 모델(GLB)은 별도 모델링 필요.
